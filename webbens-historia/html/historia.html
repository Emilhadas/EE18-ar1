Webbens Historia

World wide webb skappades av Tim Berners-Lee, på ett forsknings ceter i Schweiz, med hjälp av den första generationens HTML. Så kom världens första allmäna webbplats kom den 6 april 1991 som då var tillgänglig för alla att besöka. Efter det så har tekniken bakom hur man gör hemsidor bara blivit bättre och bättre, år 1993 så kom efterträdaren på HTML vilket var HTML 2. HTML 2 var en ny version av den tigigare HTML versionen. 1997 så kom HTML 3 vilket var ännu en förbättring av HTML 2 och var en ny version med mer uppdateringar samt mer kommand. Efter HTML 3 så kom år 1995 ett program som heter JavaScript vilket var en stor påverkan på hur man kunde göra hemsidor då man kunde göra rörliga saker och fick mer kommandond med JavaScript. Idag så finns HTML upp till verision 5.
